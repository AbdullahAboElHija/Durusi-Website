Build a full Arabic RTL frontend website called "دروسي" (Durusi) — a platform for finding Islamic religious lessons in Palestinian Arab communities inside Israel.
Tech Stack: React + Tailwind CSS + React Router
Design Requirements:

Full RTL (right-to-left) layout, Arabic typography (use Google Font: Cairo)
Color palette: Deep green (#1B4332), gold (#D4A017), warm white (#FAFAF7), soft gray (#F3F4F6)
Mobile-first, responsive design
Clean, modern Islamic aesthetic — subtle geometric patterns as decorative accents (CSS only, no images)


Pages & Components to build:
1. Homepage (الرئيسية)

Hero section with search bar: "ابحث عن درس قريب منك..." with lesson type toggle: أونلاين / في المسجد, city dropdown (only shown if "في المسجد" is selected), and topic filter
Featured upcoming lessons cards (3–4 cards), each showing a badge: 🟢 أونلاين or 🕌 في المسجد
Quick stats bar: number of lessons, mosques, cities, online lessons
CTA section: "هل أنت إمام أو مركز إسلامي؟ أضف جدول دروسك"

Lesson cards showing: title, sheikh, topic badge, type badge (أونلاين or في المسجد + city), day + time

3. Lesson Detail Page (تفاصيل الدرس)
For في المسجد lessons:

Mosque name, city (from approved list), full address, Google Maps embed placeholder

For أونلاين lessons:

Schedule (day + time)
Sheikh name, topic, description

Both types: "أضف للتذكير" button, Share button, WhatsApp contact button

🕌 درس في المسجد
💻 درس أونلاين

Mosque name, city (dropdown — Arab towns inside Israel only — full list below), address, sheikh name, lesson title, topic, day(s), time, description, WhatsApp contact

If أونلاين selected, show:

Sheikh name / page name, lesson title, topic, platform (dropdown: Zoom, YouTube, واتساب, تيليغرام, أخرى), link or channel, day(s), time, description, WhatsApp or contact link

Both: styled Arabic form, validation, success confirmation message
